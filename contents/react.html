What is JSX

JSX is an XML-like syntax extension to ECMAScript without any defined semantics. It’s NOT intended to be implemented by engines or browsers. by using JSX you can write HTML code in JavaScript then Babel transform these expressions into actual JavaScript code.It allows us to put HTML into JavaScript. JSX is like a shorthand for calling React.createElement function. Since it’s a JavaScript, we can execute JavaScript code inside JSX using {} curly-brace syntax.

Reducer

Code example of reducer

//trigger store refresh
function todoApp(state = initialState, action){
  switch (action.type) {
    case ADD_TODO:
      return Object.assign({}, state,{
      todos: [
        ...state.todos,
        {
          text: action.text,
          completed: false
        }
      ]
    })
    default:
      return state
   }
}
